WEB-APP MEMORANDUM 2.0.2 (01.09.2011).Веб-приложение "Служебные записки".

Зависимости.

Для использования данного приложения (для ее сборки и работы с текущей версией) необходимы следующие библиотеки (зависимости от внешних библиотек):

  • commons-configuration-1.6
  • commons-fileupload-1.2.1
  • commons-lang-2.4
  • javaee (только для компиляции)
  • jdb
  • jlib
  • jpersonnel
  • jstl
  • log4j-1.2.15
  • mail
  • standard

Сборка проекта.

Сборка данного проекта осуществляется с помощью сборочного скрипта ANT (используется версия 1.8.1). Описание сборочного скрипта ANT находится в виде комментариев в самом скрипте, описаны все сборочные задачи и порядок их следования, даны комментарии для сложных команд. Сборка может осуществляться как из среды IntelliJ IDEA 9.x.x и выше, так и напрямую из командной строки (без запуска среды IDE) с помощью утилиты ANT. Для сборки данной библиотеки необходимы все вышеуказанные библиотеки сторонних производителей (см. выше раздел "Зависимости").

Использование логгирования.

Для работы системы логгирования следует инициализировать логгер с именем "memorandum" - тогда все классы приложения смогут вести журналирование. Если же логгер инициализирован не будет, то при работе библиотеки будет возникать сообщение об этом (WARNING).
